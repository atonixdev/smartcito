# ORCA Logo System

This directory is the canonical source of truth for the new ORCA catfish-inspired logo pack.

Why this path is used:

- The requested `/orca/assets/logo/` location is not possible in this repository because `/orca` is already a tracked CLI launcher file.
- `assets/logo/` is the nearest valid canonical location.
- Web-served copies that must be publicly reachable are mirrored under `webapp/public/assets/logo/`.

Deliverables included here:

- `orca-catfish-auto.svg`
- `orca-catfish-dark.svg`
- `orca-catfish-light.svg`
- `orca-catfish-mono.svg`
- `orca-catfish-inverted.svg`
- `orca-catfish-outline.svg`
- `orca-catfish-app-icon.svg`
- `orca-catfish-favicon.svg`
- `orca-catfish-boot-screen.svg`
- `orca-catfish-concepts.svg`
- `orca-cli-banner.txt`

- `png/orca-catfish-dark-{256,512,1024,2048}.png`
- `png/orca-catfish-light-{256,512,1024,2048}.png`
- `png/orca-catfish-mono-{256,512,1024,2048}.png`

- `png/orca-catfish-outline-{256,512,1024,2048}.png`
- `png/orca-catfish-app-icon-{32,64,128,256,512}.png`

Recommended brand usage:

- `orca-catfish-auto.svg` or `png/orca-catfish-auto-2048.png` for the primary brand mark
- `orca-catfish-dark.svg` or `png/orca-catfish-dark-2048.png` for dark presentation surfaces
- `orca-catfish-light.svg` or `png/orca-catfish-light-2048.png` for light presentation surfaces
- `orca-catfish-app-icon.svg` or `png/orca-catfish-app-icon-512.png` for square app/icon use

Color system:

- `#000000` ORCA black
- `#003B73` ORCA ocean blue
- `#00E5FF` ORCA electric cyan
- `#FFFFFF` ORCA white
