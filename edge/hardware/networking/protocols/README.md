# Open Protocols

Protocol baseline for Orca interoperability.

## Video and Device Standards

- ONVIF for IP camera interoperability
- RTSP for live stream transport
- HTTP/2 for secure metadata and control-plane exchanges
- REST and GraphQL for platform integration

## Sensor and Telemetry Standards

- MQTT for event-driven IoT telemetry
- CoAP for constrained, low-power devices
- TLS for secure transport across all supported protocols

## Outcome

These protocols keep Orca open and extensible across vendors,
geographies, and deployment models.
