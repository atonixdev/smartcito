# Monitoring and Cooling

Hardware health monitoring for Orca deployments.

## Environmental Monitoring

- temperature sensor per rack and per hot aisle/cold aisle segment
- humidity and airflow monitoring in shared facilities
- smart cooling systems with alert thresholds
- UPS health and battery runtime dashboards

## Platform Monitoring

- Prometheus for node, container, and service metrics
- Grafana for power, temperature, network throughput, and alert views
- IDS alert feeds integrated with security dashboards

## Minimum Metrics

- rack inlet and outlet temperature
- PSU redundancy status
- disk SMART and RAID controller health
- GPU temperature, utilisation, and ECC status
- link saturation and packet loss on ingest and backbone ports
